method throwError [line 102]

	

    

    

		void throwError(
int
$errorcode, [string
$errormessage = 'An unknown error has occured.'], [string
$debugmessage = 'No messages.'])
    

    


	
		Throws an error for LiteRenderer to pickup.

It's the programmer's responsibility to not echo anything out (including any html source).  It's important to note that even though an error has been thrown, it doesn't necessarily mean that the template page stop executing. A good way to stop the execution is to return in the template page.





	
        
Parameters:

    

    

          

        int  
        $errorcode  
        The error code that will be given to the header of the return sent by the server if the template page didn't echo anything out.
      

          

        string  
        $errormessage  
        The error message that will be displayed.
      

          

        string  
        $debugmessage  
        Additional debug message.
